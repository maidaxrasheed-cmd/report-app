# 📊 Screenshot Report Generator

A simple Streamlit-based tool that generates structured PDF reports from screenshots and inputs.

---

## 🚀 Features
- Upload and process screenshots
- Generate structured PDF reports
- Clean layout rendering using ReportLab
- Simple Streamlit interface (no coding required)

---

## 🛠️ Tech Stack
- Streamlit
- Python
- ReportLab
- Pillow
